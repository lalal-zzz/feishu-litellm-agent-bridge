# 接入飞书 (Lark) 配置指南

本指南介绍如何将本地 Coding Agent 与飞书 (Lark) 进行对接，实现通过飞书渠道与 Agent 交互。

## 1. 相关资源

*   **GitHub 仓库**: [lark-coding-agent-bridge](https://github.com/zarazhangrui/lark-coding-agent-bridge)
*   **官方详细文档**: [飞书文档 - lark-channel-bridge](https://larkcommunity.feishu.cn/docx/OaRIdFIRFoLM3xxTmKwcetHqn5e)

## 2. 安装与运行

### 全局安装命令行工具

使用 `npm` 进行全局安装：

```bash
npm install -g lark-channel-bridge
```

### 启动桥接服务

在命令行中运行以下命令启动服务：

```bash
lark-channel-bridge run --skip-check-lark-cli
```

启动成功后，即可通过飞书与 Agent 进行通信。

## 3. 高级配置与管理

*   **修改默认配置**：
    配置文件路径为 `~/.lark-channel/config.json`。您可以根据需要编辑该文件以调整连接参数。
*   **查看帮助命令**：
    在终端中运行以下命令查看所有可用指令：
    ```bash
    /help
    ```
